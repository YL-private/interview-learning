# 组件按需加载

1.路由懒加载

```javascript
// 原来
import Home from './Home.vue'
// 改成
const Home = () => import('./Home.vue')

const routes = [
  { path: '/', component: Home }
]
```

2.组件局部加载

```javascript
import { defineAsyncComponent } from 'vue'

// 定义：按需加载组件
const DialogComp = defineAsyncComponent({
  loader: () => import('./DialogComp.vue'),
  loadingComponent: () => import('./Loading.vue'), // 加载中
  errorComponent: () => import('./Error.vue'),     // 加载失败
  delay: 200,
  timeout: 5000
})
```
