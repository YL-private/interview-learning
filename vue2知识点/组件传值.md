## 父子组件传值
1. props $emit
2. .sync修饰符可以省略多余的事件
子组件：
<script>
  props: {
    name: {
      type: String,
      default: 'x'
    }
  }
  methods: {
    change() {
      this.$emit('update: name', 'xyl')
    }
  }
</script>
父组件
<child :name.sync="name"><child>
