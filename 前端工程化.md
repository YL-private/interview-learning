# 前端工程化六大模块

## 规范化

1. eslint代码校验
2. prettier 代码格式化
3. git commit提交规范
4. 命名规范，目录规范，接口规范

## 模块化

1. js模块化: ESModule或者CommonJS
2. css模块化: Sass/Less
3. 组件模块化

## 组件化

页面拆分为可复用组件

## 构建打包

开发: vite/webpack本地启动、热更新
生产: 压缩、treeShaking、代码分割、按需加载

## 自动化

1. 自动打包、自动测试
2. CI/CD自动部署

## 质量与部署

1. 单元测试
2. 接口测试、性能监控
3. 灰度发布、版本管理
